# Streamlit Weight Converter ⚖️

Aplikasi web sederhana untuk konversi berat yang dibangun menggunakan Streamlit.

## 💡 Fitur

- Konversi antara satuan berat:
  - Kilogram (kg)
  - Gram (g)
  - Pound (lb)
  - Ounce (oz)
