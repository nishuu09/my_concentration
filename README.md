# Web Konversi Berat ⚖️

Aplikasi web sederhana menggunakan Flask untuk mengonversi berat dari satu satuan ke satuan lainnya.

## 🔧 Fitur

- Konversi antara:
  - Kilogram (kg)
  - Gram (g)
  - Pound (lb)
  - Ounce (oz)
